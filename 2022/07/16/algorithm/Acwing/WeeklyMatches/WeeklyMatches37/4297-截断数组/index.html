4297-截断数组

发表于2022-07-16|更新于2022-07-16|算法AcwingAcwing周赛
|字数总计:692|阅读时长:2分钟|阅读量:

题目

给定一个长度为 n 的数组 d1,d2,…,dnd_1,d_2,…,d_nd1​,d2​,…,dn​。

现在，要将该数组从中间截断，得到三个子数组（可以为空）。

不妨设第一个子数组包含 a 个元素，第二个子数组包含 b 个元素，第三个子数组包含 c 个元素。

那么三个子数组的各元素之和 sum1,sum2,sum3 依次为：

sum1=∑1≤i≤adi，sum2=∑a+1≤i≤a+bdi，sum3=∑a+b+1≤i≤a+b+cdi。sum1=\sum_{1≤i≤a}{d_i}，sum2=\sum_{a+1≤i≤a+b}d_i，sum3=\sum_{a+b+1≤i≤a+b+c}d_i。sum1=∑1≤i≤a​di​，sum2=∑a+1≤i≤a+b​di​，sum3=∑a+b+1≤i≤a+b+c​di​。

注意，空数组的各元素之和为 0。

我们希望截断后的三个子数组满足：

  • sum1=sum3。
  • 满足上一条件的情况下，sum1 尽可能大。
    请你计算并输出 sum1 的最大可能值。

显然，本题一定有解，因为可以令 a=0,b=n,c=0。

输入格式

第一行包含整数 n。

第二行包含 n 个整数 d1,d2,…,dnd_1,d_2,…,d_nd1​,d2​,…,dn​。

输出格式

输出一个整数，表示 sum1 的最大可能值。

数据范围

前 6 个测试点满足 1≤n≤101≤n≤101≤n≤10。
所有测试点满足 1≤n≤2×105，1≤di≤1091≤n≤2×10^5，1≤d_i≤10^91≤n≤2×105，1≤di​≤109。

分析

  1. sum1其实就是数组前a个数的和，sum3就是数组后c个数的和，那么我们就可以想到把前缀和先算好，再来进一步的比较。
  2. 由于sum可能达到1×10141×10^{14}1×1014，所以考虑用unsigned long long来储存。
  3. 接下来就要查找前缀和了，可以考虑使用双指针,l和r,由于数组中都是非负整数，所以当集合内元素增加时，sum是非严格单调递增的。所以当sum1<sum2的时候,就需要增加左边元素个数，sum1>sum2的时候就需要增加右边元素个数，直到sum1=sum2就更新sum1的最大值,直致l!<r。
